# Bloop

### Run
```
bloop <file>
```

Includes current path in include, as well as path to stdlib.

### Install Package
```
bloop install <file>
```

Includes current path in include, as well as path to stdlib.

### Build
```
bloop install <file>
```

Builds from file, fails if no main found in file. Creates a blam.json file. Creates build dir.
